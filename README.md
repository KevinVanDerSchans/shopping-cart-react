## Features

<div>

    ✔️ Filter by price (0€ - 1000€)

    ✔️ Filter by category (All, Laptops and Smartphones)

    ✔️ Add product to cart

    ✔️ Remove product

    ✔️ Add more quantities to cart

    ✔️ Clear shopping cart.

</div>
